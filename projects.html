Space Raiders • Aug 2025 – Present

APEX Flight Computer

The APEX flight computer is a custom ESP32-S3 based embedded system designed to provide robust flight control and telemetry capabilities for high-altitude rocketry applications. The primary goal of APEX is to operate the Airbrakes system, enabling precise altitude control during descent through real-time adjustments based on sensor data using a PID loop. ![APEX Block Diagram](images/apex/photos/block-diagram.png) - Key Features: - Redundant sensor suites and stackup tuned for signal integrity; board bring-up and validation. - Real-time PID control loops using simulation-driven parameters to support airbrake actuation. - FreeRTOS scheduling, dual-core partitioning, and telemetry integration for flight logging. - Sensors: - Barometer - Accelerometer - Magnetometer - 6-DoF IMU - GPS
